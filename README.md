This is the code executing modelling of Pd-Cu-S systematics for the manuscript "Fluid-mediated PGE enrichment of magmatic low-sulfide ores as modelled for the Norilsk deposits" by Chayka et al. 
Please, see Text S2 for details of calculation.
This script is designed to launch in desktop Spyder 5.
